# 猫门智能排版器 Pro

一款专为公众号创作者设计的智能排版工具，结合 AI 能力，帮助你快速完成文章排版和营销物料生成。

## 功能特点

### 四种创作模式

| 模式 | 适用场景 |
|------|----------|
| **模式A：文章排版** | 文章已写好，需要添加金句、分段、强调等排版格式 |
| **模式B：主题创作** | 只有主题或想法，需要 AI 生成完整爆款文章 |
| **模式C：长稿提炼** | 有课程逐字稿、播客文字稿等，需要提炼成精华文章 |
| **模式D：素材整合** | 有多个碎片素材，需要整合成原创文章 |

### 智能排版

- **金句提炼** - 自动识别并高亮展示精彩语句
- **结构优化** - 智能分段、添加小标题
- **配图建议** - 根据内容生成 AI 绘图提示词
- **多种配色** - 莫兰迪、森绿、薰衣草、奶茶、黑金、珊瑚

### 营销物料一键生成

- 朋友圈文案（作者版/转发版）
- 小红书图文 + 标题备选
- 社群转发话术
- 私聊推荐话术
- 文章标题备选
- 目标人群画像
- 金句卡片文案
- 短视频口播文案
- SEO 关键词

### 风格预设

**预设风格**：猫门风格、柠檬式、治愈式、干货式、故事式

**博主风格**：咪蒙式、开智/阳志平式、知乎大V式、生活方式式

支持自定义风格描述或参考文章学习

## 使用方法

1. **选择模式** - 根据你的素材情况选择合适的创作模式
2. **生成提示词** - 复制提示词发送给 AI（推荐 Claude / Gemini）
3. **编辑排版** - 粘贴 AI 返回的内容，智能解析后可视化编辑
4. **生成物料** - 一键生成全套营销物料

## 在线使用

直接打开 HTML 文件即可使用，无需安装任何依赖。
